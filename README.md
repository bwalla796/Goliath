# Goliath

## A simple ASCII rougelike game for your CLI

### Why did I build Goliath?

I enjoy all kinds of rougelike games and I like working in the command line, I figured why not combine them and build my first game?

Updates in pipeline are as follows; relics after boss fights, combat animations, monster and boss sprites.

### Quick Start

Requires:
    Python: 3.12

Clone down this repository and run ./build.sh from the root of the repository. Enjoy!
